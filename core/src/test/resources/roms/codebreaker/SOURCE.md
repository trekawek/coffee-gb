# CodeBreaker rumble demo

`In-Game Rumble Demo (PD) [C].gbc` is the public-domain demo attached to
[coffee-gb issue #240](https://github.com/trekawek/coffee-gb/issues/240). It writes
the CodeBreaker pass-through accessory's motor sequence through HRAM address `FFFE`.
